# ESP32_CAM_UPD_SERVER
ESP32_CAM模块，利用UPD协议实时传输图像，基于Arduino实现


![image](https://user-images.githubusercontent.com/58870893/204182474-693a78bd-fac9-484c-955e-9de03c7a891d.png)

## 项目特性
- 利用`Pygame`实现接收端界面，代码简洁，第三方库依赖少
- 提供了两种UPD服务的连接方案
- 基于多线程，失去连接时界面不会卡死
- 注释丰富，适合学习
